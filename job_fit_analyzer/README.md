# Job fit analyzer

A Python project that compares a resume to a job description and calculates a match score based on relevant keywords.

## Features

-Reads resume text from a file
-Reads job description from a file
-Finds matching keywords
-Shows missing keywords
-Calculates a fit score
-Reccomends whether to apply

## Skills used

-Python
-Functions
-Loops
-Lists
-Conditionals
-File Handling
-String matching

## Files
main.py
resume.txt
job.txt

# How to run

1. Add your resume text to resume.txt
2. Add a job description to job.txt
3. Run:

python main.py
